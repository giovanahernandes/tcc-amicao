#  Amicão — Plataforma Digital

##  Sobre o Projeto
O **Amicão** é uma plataforma digital desenvolvida como **Trabalho de Conclusão de Curso (TCC)**, com o objetivo de oferecer uma solução tecnológica voltada para **tutores de animais de estimação** que buscam serviços confiáveis para o cuidado de seus pets.

A plataforma permite centralizar informações e serviços como **agendamento de passeios, hotéis para pets, comparação de preços de ração e outros serviços relacionados ao bem-estar animal**, promovendo praticidade, economia e segurança.

A ideia do projeto surgiu a partir da dificuldade de encontrar **serviços acessíveis e confiáveis** voltados para animais de estimação em um único ambiente digital.

---

## Objetivo
Desenvolver uma aplicação web que:
- Atenda uma necessidade real
- Utilize conceitos aprendidos durante o curso
- Integre front-end e lógica de programação
- Apresente organização e boa estrutura de sistema
- Priorize segurança e confiabilidade

---

##  Funcionalidades
- Navegação entre páginas
- Exibição de serviços voltados para cuidados com animais
- Interface simples e intuitiva
- Estrutura preparada para futura integração com banco de dados

---

##  Público-Alvo e Segurança
A plataforma é destinada a:
- Pessoas que possuem animais de estimação
- Profissionais e prestadores de serviços com familiaridade com animais
- Pessoas com formação ou cursos na área animal, como **Medicina Veterinária** e áreas relacionadas

Essa definição de público-alvo visa **aumentar a segurança e a confiabilidade dos serviços oferecidos**, garantindo que os usuários tenham afinidade e responsabilidade no cuidado com os animais.
